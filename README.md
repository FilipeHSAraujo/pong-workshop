# Pong Workshop - Aprenda Pygame do Zero

Projeto base para o workshop de introdução ao Pygame.

## Como usar

1. Instale o Pygame
```bash
pip install pygame
```

2. Rode o jogo
```bash
python main.py
```

3. Abra o arquivo `main.py` e vá descomentando as seções na ordem dos comentários/desafios.
